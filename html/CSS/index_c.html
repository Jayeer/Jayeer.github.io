Структура CSS кода

Стандартная структура CSS кода выглядит вот так:

                    селектор {
                        свойство: значение;
                    }

А если говорить простым языком , то так

                    объект {
                        параметр: значение;
                    }

Справочник по HTML

Селекторы

Селектор указывает на какой либо объект.

В виде селектора может выступать класс , он указывает на класс и записывается так:


.название_класса

Что-бы обозначить класс тега в HTML нужно внутри него написать class="Название класса"

Также как селектор можно использовать id. Запись:

#id

Что-бы обозначить id тега в HTML нужно внутри него написать id="id тега"

Справочник по HTML

Селекторы можно перечислять через запятю.Например:

                    A , B , C {

                    }

Также в роли селектора можно использовать знак *, он значает что свойства применятся ко свем тегам на старнице.

Псевдоклассы

К селектору также можно преписать псевдокласс , таких классов есть несколько:

  • hover - означает что свойства будут применены к ссылке когда на неё наведен курсор мыщи.
  • visited - означает что свойства будут применены к просмотренной ссылке
  • active - означает что свойства будут применены к ссылке в момент нажатия.
  • nth-child(n) - означает что свойства будут применены к дочернему(вложенному) тегу под порядковым номером n.

Синтаксис:

                    тег:псевдокласс {

                    }

Свойства

Свойства отвечают за разные параметры элементов , напрмер размер , цвет или позицию.

Свойства можно перечислять с ряд . но гораздо удобней делать это в столбик.

После каждого свойства ставится точка с запятой (;).

Теперь самые часто используемые свойства

Текст

  • color - отвечает за цвет текста.
  • font-family - шрифт.
  • line-height - расстояние между строк.
  • font-size - размер шрифта.

Фон

  • background - фон.
  • background-color - цвет фона.

Позиции элементов

  • margin - отступ со всех сторон.
  • margin-left | right | bottom | top - отступ с орпеделенной стороны.
  • text-align - сторона расположения текста (например справа или снизу).
  • float - сторона расположения элемента.

Другие свойства

  • transform - исполбзуется для трансформации объекта (например увеличения , вращения , движения и т.д.)
  • height - высота
  • width - ширина
  • position - позиция (например фиксированая или изменяемая).
