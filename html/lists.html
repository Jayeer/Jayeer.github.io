Списки

Тег ul нужен для создания ненумерованного списка.

Тег li нужен для добавления элементов в список.

Тег ol нужен для создания нумерованного списка.

Есть 3 тега для обозначения терминов: dl, dd и dt

Тег <dl> - это контейнер.

Тег <dd> - создаёт термин,

А тег <dt> добавляет определение термина.

Атрибут ненумерованного списка type="disc" добавляет диски перед каждым элементом списка.

Атрибут ненумерованного спискаtype="circle" добавляет круг перед каждым элементом списка.

Атрибут ненумерованного спискаtype="square" добавляет квадрат в начале каждого элемента списка.

Нумерованные списки

Атрибут нумерованного списка type="1" добавляет арабские цифры в начале каждого элемента списка.

Атрибут нумерованного списка type="I" добавляет заглавные римские цифры в начале каждого элемента списка.

Атрибут нумерованного списка type="i" добавляет строчные римские цифры в начале каждого элемента списка.

Атрибут нумерованного списка reversed добавляет перед элементами нумерацию в обратном порядке.

Атрибут нумерованного списка type="A" добавляет заглавные латинские буквы в начале каждого элемента списка.

Атрибут нумерованного списка type="a" добавляет строчные латинские буквы в начале каждого элемента списка.
