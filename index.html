Тег: <!DOCTYPE html> - нужен для обозначения что сайт написан на html

Тег: <pre> - нужен для вывода текста с сохранением форматирования

Тег: <html> нужен для обозначения начала и конца страницы

Атрибуты - это параметры тега

Атрибут тега <html> lang="ru" - это язык на котором будет индексироватся сайт в ПСе

Одинарный тег - это тег который имеет начало и не имеет конца

Справочник по CSS

Двойной(закрывающий) тег - это тег который имеет открывающий и закрывающий тег т.е. начало и конец

Тег: <head> - нужен для настройки сайта

Вложенный тег <meta> - нужен для указания метаданных

Вложенный тег <title> - нужен для обозначения названия сайта

Тег <Header> - нужен для создания шапки сайта

Вложенный тег <H1> - нужен для создания заголовка

Тег: <body> - нужен для создания тела (основной части) сайта

Вложенный тег <P> - нужен для создания параграфа

Тег <img> добавляет на страницу картинку

Атрибут тега <img> src означает ссылку на картинку.

Атрибут тега <img> alt означает альтернативный текст который показывается если картинка не может быть отображена на странице.

Атрибут тега <img> height указывает высоту картинки.

Атрибут тега <img> width указывает ширину картинки.

Тег <a> используется для добавления текста на сайт с возможностью создания ссылки.

Атрибут тега <a> href указывает адрес ссылки.

Знак # добавляется в адрес ссылки перед названием страницы и означает якорную ссылку и указывает на определенный элемент на странице.
